# Student Grade Management System

## Project Purpose
This is a Java console-based application developed using Object-Oriented Programming (OOP) concepts.

The system allows teachers to:
- Add student records
- Display all students
- Search students by ID
- Calculate average marks

## How to Run

1. Compile the Java files:

javac Student.java StudentGradeManagementSystem.java

2. Run the program:

java StudentGradeManagementSystem
